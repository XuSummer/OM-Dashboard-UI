# OpenMediation Dashboard UI

### Project setup

- Clone repo
```shell script
git clone https://github.com/AdTiming/OM-Dashboard-UI.git
cd OM-Dashboard-UI
```

- Install dependencies
```shell script
yarn install
```

- Compiles and hot-reloads for development
```shell script
yarn run serve
```

- Compiles and minifies for production
```shell script
yarn run build
```

- Lints and fixes files
```shell script
yarn run lint
```
